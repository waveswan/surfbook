# surfbook

Русскоязычный перевод регламента ISA по сёрфингу.

Проект TopHeats — система судейства соревнований в реальном времени.
https://topheats.ru

## Глоссарий

Утверждённая русская терминология хранится в файле [`isa_surfing_terms_ru.yml`](isa_surfing_terms_ru.yml).

Принцип: **один термин = одно значение**. Не изобретай синонимы и не меняй утверждённые переводы.

Примеры:
- heat → хит
- interference → помеха
- wave count → лимит волн
- penalty → штраф
- priority → приоритет
